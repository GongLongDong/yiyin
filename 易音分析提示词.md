# 易音语言学讨论提示词规则体系（完整实例库版）

## 一、核心音素实例库

### 1.1 基础音素实例
```
【a音实例库】
- 音源场景：婴儿哭声"wu a wu a"
- 发音特征：嘴巴张开，形成开口状态
- 多重含义：距离、空间、开、施压、变化、量
- 验证单词：max, mass, ass, gap, act
- 易音公式：ga = ge + a, au = a + wu

【wu音实例库】
- 音源场景：婴儿哭声"wu a wu a"
- 发音特征：嘴唇闭合，形成闭合状态
- 多重含义：闭合、封闭
- 验证单词：mouth中的au = a + wu
- 易音公式：au = a + wu

【m音实例库】
- 音源场景：婴儿需求食物时的口型音"ma m ma m"
- 发音特征：嘴唇始终闭合，发音时带鼻音，气流从鼻腔流出
- 多重含义：末端、表面、闭合、边界
- 验证单词：mouth, max, mass, mine, mum
- 易音公式：mae = me + ae, mum = m + u/ʌ/ + m

【e音实例库】
- 音源场景：打嗝音
- 发音特征：发/ə/音，表示闭合状态
- 多重含义：闭合、收敛、聚焦、顶部、核
- 验证单词：mine, see中的ee
- 易音公式：ee = e + e

【s音实例库】
- 音源场景：蛇音，蛇吐舌头的方式来感知外部环境信息
- 发音特征：蛇吐舌头感知
- 多重含义：感知、观察
- 验证单词：sea, see
- 易音公式：sea = s + ea, see = s + ee

【h音实例库】
- 音源场景：追捕猎物时的喘气音"hei"
- 发音特征：剧烈运动时的喘气
- 多重含义：指向、追踪、方向性
- 验证单词：hunt, head, hei
- 易音公式：hei = he + ei
```

### 1.2 组合音素实例
```
【ae音实例库】
- 音源场景：羊叫声
- 发音特征：羊叫声的元音，发/æ/音
- 多重含义：慢、漫延、晏（晚了、迟了）
- 验证单词：mae【慢】, mae【漫】
- 易音公式：mae = me + ae

【ei音实例库】
- 音源场景：追捕猎物时的喘气音"hei"
- 发音特征：剧烈运动时的喘气
- 多重含义：量、够、达到
- 验证单词：hei【候】
- 易音公式：hei = he + ei, gei = ge + ei

【au音实例库】
- 音源场景：mouth中的组合音
- 发音特征：au = a + wu，开到闭的咀嚼运动
- 多重含义：开到闭的运动
- 验证单词：mouth
- 易音公式：au = a + wu

【ea音实例库】
- 音源场景：波的形态
- 发音特征：e（顶部、收敛点）+ a（底部、扩展区域）
- 多重含义：一个周期的波的形态
- 验证单词：peak, sea
- 易音公式：peak = p + ea + k, sea = s + ea

【ee音实例库】
- 音源场景：双聚焦点
- 发音特征：e + e，两个收敛点、两个聚焦点
- 多重含义：两个收敛点、两个聚焦点
- 验证单词：see
- 易音公式：see = s + ee
```

## 二、易音公式实例库

### 2.1 基础组合公式
```
【ga = ge + a】
- ge：具备、拥有
- a：距离、空间
- ga：在两者之间、媒介、界
- 验证：gap中的ga体现"具备距离"的含义

【mae = me + ae】
- me：末端、表面
- ae：羊叫声的元音
- mae：慢、漫延
- 验证：mae【慢】= me【末】+ ae【晏】

【hei = he + ei】
- he：追踪状态
- ei：量
- hei：等候、期望
- 验证：hei【候】= he【核】+ gei【够】
```

### 2.2 单词分解公式
```
【mouth = m + au + th】
- m：末端/表面（嘴唇和牙齿的末端边界）
- au：au = a + wu，表示开到闭的咀嚼运动
- th：待分析
- 整体含义：嘴巴的开闭特征

【max = m + a + x】
- m：末端/表面
- a：距离
- x：待分析
- 整体含义：最大，即"末端之间的距离达到最大"

【mass = m + a + ss】
- m：末端/表面
- a：距离
- ss：重复、往复、迭代或一对、两个、双
- 整体含义：质量/团块，即"末端之间有一定距离的聚集物"

【ass = a + ss】
- a：距离
- ss：重复、往复、迭代或一对、两个、双
- 整体含义：屁股，即"两个末端之间有距离的部位"

【mine = m + in + e】
- m：末端/表面（地表）
- in：在...里面、内部
- e：闭合状态（掩埋、隐藏）
- 整体含义：矿，即"被掩埋在地表内部的矿物"

【mum = m + u/ʌ/ + m】
- m：上下嘴唇的末端
- u/ʌ/：从闭合到张开的状态变化
- m：上下嘴唇的末端（再次闭合）
- 整体含义：妈妈，即"通过嘴巴张开闭合的行为引导婴儿喂食的母亲"

【gap = ge + a + p】
- ge：具备/拥有
- a：距离/空间/量
- p：待分析
- 整体含义：间隙、隔阂，即"存在距离/空间"

【peak = p + ea + k】
- p：位置或方位信息
- ea：一个周期的波的形态
- k：聚焦点，收敛点的位置
- 整体含义：山峰的尖顶

【sea = s + ea】
- s：感知
- ea：一个周期的波的形态
- 整体含义：海洋，即"通过感知 + 一个周期的波的形态 = 感知到的波浪运动水域"

【see = s + ee】
- s：感知
- ee：两个收敛点、两个聚焦点
- 整体含义：看见，即"通过感知 + 两个收敛点、两个聚焦点 = 用眼睛聚焦感知"

【act = a + ct】
- a：运动/动作
- ct：待分析
- 整体含义：行动、行为，即"产生运动/动作"

【hunt = h + unt】
- h：指向、方向性
- unt：待分析
- 整体含义：追踪、打猎，即"指向目标的运动"

【head = h + ea + d】
- h：指向、方向性
- ea：一个周期的波的形态
- d：待分析
- 整体含义：头部，即"具有指向功能的头部"
```

## 四、推测验证实例库

### 4.1 字母音≠实际发音规律
```
【推测内容】：字母音≠实际发音表示状态变化，表述运动

【验证实例】：
1. mouth：ou（字母）≠ au（发音）→ 表示状态变化 → 表述嘴巴运动
2. mum：u（表音：闭合）≠ /ʌ/（实音a：张开）→ 表示状态变化 → 表述从闭合到张开的变化
3. gap：a（表音/ei/）≠ /æ/（实音ae）→ 表示状态变化 → 表述状态变化

【验证结果】：推测得到验证，规律成立
```

### 4.2 同概念多音现象

【验证实例】：
- 闭合概念：
  - wu音：来自婴儿哭声，嘴唇闭合状态
  - e音：来自打嗝音，表示闭合状态
- 距离概念：
  - a音：来自婴儿哭声，嘴巴张开

【验证结果】：推测得到验证，同概念多音现象存在

## 八、易音系统理论基础

### 8.1 易音系统的三个基础
```
【有限性】
- 自然界音的数量是有限的
- 人类需要重复利用现有的音来表达不同的概念
- 音的重用和组合是语言发展的必然

【可采纳性】
- 模仿的准确性
- 确保音在传递过程中保持其基本特征
- 音的特征稳定性是易音系统的基础

【共识性】
- 群体认知的形成
- 古代基于共同经历
- 现代基于教育传承
```

### 8.2 易音双向运算
```
【正向组合】
- 两个或多个音组合形成新的音
- 例：ge + a = ga
- 信息组合机制：每个音素承载特定信息，音素组合形成更复杂的信息

【反向分解】
- 一个音可以分解为多个音
- 例：ga = ge + a
- 分解分析：通过分解理解音的内部构成

【信息组合机制】
- 每个音素承载特定信息
- 音素组合形成更复杂的信息
- 组合后的含义是各音素含义的综合
```
